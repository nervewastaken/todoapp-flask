# Documentation

## steps to start 
### 1) Fork
### 2) create a virtual environment in the todoapp folder  
### 3) cd client
### 4) npm i (install packages from package-json
### 5) go to todoapp/src -> run command python main.py
### 6) go to todoapp/client -> run command npm start
